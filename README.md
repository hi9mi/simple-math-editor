# Simple-math-editor

### This is a small application with which you can:

- Create mathematical expression with description

# Available scripts

## In the project directory, you can run:

### Runs the app in the development mode.

```shell
pnpm dev
```

### Builds the app for production to the build folder.

```shell
pnpm build
```

### Runs locally preview production build.

```shell
pnpm preview
```

### Runs eslint.

```shell
pnpm lint
```

## How to run locally

### Clone repo:

```shell
git clone https://github.com/hi9mi/simple-math-editor.git
```

### Then install dependencies:

```shell
pnpm i
```

### And then run script:

```shell
pnpm dev
```
